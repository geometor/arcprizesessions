def transform_grid(grid):
    rows, cols = grid.shape

    for i in range(rows):

            if grid[i, j] == 8:

The transformation can be described as follows: For each light blue (8) cell in the input grid:

  3. If both conditions are true (adjacent to white and outermost in cluster), change the light blue (8) cell to blue (1). Otherwise, leave the cell unchanged.
